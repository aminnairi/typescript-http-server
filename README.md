# typescript-http-server

Typescript library for declaratively creating a HTTP server.

## Requirements

- [Node.js](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)

## Installation

```console
$ npm install typescript @types/node @aminnairi/typescript-http-server
```
